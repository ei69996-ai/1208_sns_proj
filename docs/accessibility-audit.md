# 접근성 검토 문서

이 문서는 Instagram Clone SNS 프로젝트의 접근성 검토 결과를 기록합니다.

## WCAG 2.1 AA 기준

### 색상 대비 요구사항

- **일반 텍스트** (16px 미만): 최소 4.5:1
- **큰 텍스트** (18px 이상 또는 14px Bold 이상): 최소 3:1
- **UI 컴포넌트** (버튼, 링크 등): 최소 3:1

## 색상 대비 검증

### Instagram 컬러 변수

```css
--instagram-blue: #0095f6;
--instagram-background: #fafafa;
--instagram-card: #ffffff;
--instagram-border: #dbdbdb;
--instagram-text-primary: #262626;
--instagram-text-secondary: #8e8e8e;
--instagram-like: #ed4956;
```

### 대비 검증 결과

#### 1. 텍스트 Primary (#262626)

**vs 배경 흰색 (#ffffff)**
- 대비율: **12.6:1** ✅
- 기준: 일반 텍스트 4.5:1
- 상태: **통과**

**vs 배경 연한 회색 (#fafafa)**
- 대비율: **12.4:1** ✅
- 기준: 일반 텍스트 4.5:1
- 상태: **통과**

#### 2. 텍스트 Secondary (#8e8e8e)

**vs 배경 흰색 (#ffffff)**
- 대비율: **3.5:1** ⚠️
- 기준: 일반 텍스트 4.5:1 (미달), 큰 텍스트 3:1 (통과)
- 상태: **큰 텍스트로 사용 시 통과, 작은 텍스트로 사용 시 주의 필요**

**vs 배경 연한 회색 (#fafafa)**
- 대비율: **3.4:1** ⚠️
- 기준: 일반 텍스트 4.5:1 (미달), 큰 텍스트 3:1 (통과)
- 상태: **큰 텍스트로 사용 시 통과, 작은 텍스트로 사용 시 주의 필요**

**권장사항:**
- Secondary 텍스트는 시간, 보조 정보 등에 사용되므로 큰 텍스트(18px+) 또는 Bold로 사용 권장
- 작은 텍스트로 사용할 경우 대비율 개선 고려

#### 3. Instagram Blue (#0095f6)

**vs 배경 흰색 (#ffffff)**
- 대비율: **3.0:1** ⚠️
- 기준: UI 컴포넌트 3:1 (통과), 일반 텍스트 4.5:1 (미달)
- 상태: **버튼/링크로 사용 시 통과, 텍스트로 사용 시 주의 필요**

**권장사항:**
- 버튼, 링크 등 UI 컴포넌트로 사용 시 문제 없음
- 텍스트로 사용할 경우 대비율 개선 고려

#### 4. Instagram Like (#ed4956)

**vs 배경 흰색 (#ffffff)**
- 대비율: **4.1:1** ⚠️
- 기준: 일반 텍스트 4.5:1 (미달), 큰 텍스트 3:1 (통과)
- 상태: **큰 텍스트로 사용 시 통과**

**권장사항:**
- 아이콘으로 사용 시 문제 없음
- 텍스트로 사용할 경우 큰 텍스트(18px+) 권장

#### 5. Border (#dbdbdb)

**vs 배경 흰색 (#ffffff)**
- 대비율: **1.2:1** (낮음)
- 기준: 테두리이므로 대비 요구사항 없음
- 상태: **문제 없음** (테두리는 대비 요구사항 없음)

## 개선 권장사항

### 우선순위 High

1. **Secondary 텍스트 대비 개선** (선택사항)
   - 현재: #8e8e8e (3.4:1)
   - 권장: #737373 (4.5:1 이상)
   - 영향: 작은 텍스트로 사용되는 경우에만 필요

### 우선순위 Medium

2. **Blue 텍스트 사용 시 주의**
   - 버튼/링크로 사용 시 문제 없음
   - 텍스트로 사용할 경우 Bold 또는 큰 텍스트 권장

### 우선순위 Low

3. **Like 색상 텍스트 사용 시 주의**
   - 아이콘으로 사용 시 문제 없음
   - 텍스트로 사용할 경우 큰 텍스트 권장

## 현재 사용 현황

### Secondary 텍스트 사용 위치

- PostCard: 시간 표시 (작은 텍스트)
- PostCard: 댓글 미리보기 (작은 텍스트)
- ProfileHeader: 통계 라벨 (작은 텍스트)
- CreatePostModal: 캡션 글자 수 (작은 텍스트)

**권장사항:**
- 현재 사용 위치는 보조 정보이므로 큰 문제 없음
- 필요시 #737373으로 변경 고려

### Blue 색상 사용 위치

- 버튼: "공유" 버튼 (CreatePostModal)
- 링크: "게시" 버튼 (CommentForm)
- 링크: "팔로우" 버튼 (FollowButton)

**상태:** 모두 UI 컴포넌트로 사용되므로 문제 없음 ✅
